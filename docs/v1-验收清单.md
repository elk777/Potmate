# Potmate v1 验收清单

## 核心闭环

- 首次打开显示空花盆。
- 写下一条记录后生成植物，植物名称、叶形、颜色会随文本变化。
- 连续记录多天后植物进入发芽、幼苗、成株、花苞、绽放阶段。
- 长时间未记录时活力下降，重新记录后恢复。
- 绽放后可以结种收藏，图鉴中出现已收藏植物。
- 选择种子后再记录，新植物继承上一代的部分形态。

## 桌面能力

- macOS `.app` 可以启动，窗口透明、置顶、无边框。
- 托盘菜单可显示、隐藏、退出应用。
- 设置中可开启每日提醒，系统通知权限被请求。
- 设置中可开启开机自启，桌面端状态可见。
- 键鼠养分开启时，应用窗口内活动会计入养分。
- 授予系统辅助功能权限后，后台键鼠活动会计入养分。
- 未授予系统权限时，全局监听降级为窗口内活动，设置中显示状态。

## 数据与备份

- 关闭重开后植物、记录、图鉴、设置仍然保留。
- 旧 `localStorage` 数据会迁移到 Tauri Store。
- 导出备份会生成 `potmate-backup-YYYY-MM-DD.json`。
- 导入有效备份后当前花园状态恢复。
- 导入无效 JSON 或非 Potmate 备份不会破坏当前数据。

## 打包验证

```bash
pnpm build
pnpm build:desktop
pnpm package:dmg
```

通过后应生成：

- `src-tauri/target/debug/bundle/macos/Potmate.app`

如需生成 debug DMG，可在 `.app` 构建成功后运行：

```bash
pnpm package:dmg
```

## 已知系统边界

- macOS 全局键鼠监听需要用户在系统设置中授予辅助功能或输入监控相关权限。
- 当前 Windows 包需要在 Windows 机器上实际运行验证托盘、自启和全局监听。
- Debug 包可用于 macOS 自用/内部测试；未签名、未公证时首次打开可能需要手动允许。
- 当前未配置自动更新。
