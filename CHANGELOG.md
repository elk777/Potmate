# Changelog

All notable Potmate changes are documented here.

## 0.1.0 - 2026-07-04

### Added

- Desktop flowerpot companion built with Tauri 2, Vue 3, TypeScript, Rust, and Canvas.
- Daily note driven plant generation and growth.
- Bloom, seed inheritance, gallery, and backup import/export.
- Mini record bubble, control dock, dewdrop action menu, and pass-through mode.
- Tray menu, notifications, optional autostart, and optional activity nourishment.
- GitHub Releases preparation for macOS, Windows, and Linux installers.

### Known Limits

- macOS and Windows packages are unsigned unless release secrets are configured.
- Linux transparent window behavior can vary by desktop environment.
- Auto-update is not enabled.
