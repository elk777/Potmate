# Changelog

All notable Potmate changes are documented here.

## 0.1.0 - 2026-07-04

### 新增

- 基于 Tauri 2、Vue 3、TypeScript、Rust 和 Canvas 的桌面花盆桌宠。
- 每日记录驱动植物生成和成长。
- 支持盛放、结种、图鉴、种子继承、备份导入和导出。
- 支持最小态记录气泡、控制台抽屉、露珠花环动作菜单和画布穿透。
- 支持托盘、通知、可选开机自启和可选活动养分。
- 准备 GitHub Releases 的 macOS、Windows 和 Linux 安装包分发。

### 已知限制

- macOS 和 Windows 安装包暂未签名。
- Linux 透明窗口表现会受桌面环境影响。
- 当前版本未启用自动更新。
