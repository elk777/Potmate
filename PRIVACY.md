# Potmate 隐私政策

最后更新：2026-07-04

Potmate「小盆友」是本地优先的桌面花盆桌宠。当前版本不需要账号，不提供云同步，也不会把你的每日记录上传到服务器。

## 我们保存什么

Potmate 会在本机保存：

- 每日记录文本。
- 植物状态、成长进度、图鉴和种子。
- 应用设置，例如提醒、开机自启、温和模式等。
- 用户手动导出的 JSON 备份文件。

## 我们不做什么

Potmate 当前不会：

- 上传每日记录。
- 上传植物状态或设置。
- 收集账号、邮箱或手机号。
- 读取浏览器内容、聊天内容或其他应用窗口内容。
- 记录具体按键值、密码或鼠标坐标。
- 运行广告、追踪 SDK 或第三方统计 SDK。

## 键盘和鼠标活动

Potmate 的“活动养分”是可选功能，用于判断你是否仍在电脑前活动，让植物获得温和的辅助成长信号。

该功能只统计事件类型和次数，例如“发生了一次点击”或“发生了一次按键”。它不会保存你按下了哪个键，不会保存鼠标位置，也不会读取你正在输入的内容。

如果系统权限不足，Potmate 会降级为应用窗口内活动监听。

## 通知

Potmate 可能发送每日记录提醒。通知内容不包含你的记录原文。你可以在应用设置中关闭提醒。

## 开机自启

Potmate 只会在你主动开启后设置开机自启。关闭开关后，应用会尝试取消系统启动项。

## 文件访问

Potmate 通过系统文件选择器导入和导出 JSON 备份。应用不会主动扫描你的文件系统。

## 数据删除

当前版本没有云端账户。你可以通过应用内备份/重置能力管理本地数据，也可以卸载应用并删除系统应用数据目录。

## 联系

如发现隐私或安全问题，请通过 GitHub Issues 或 `SECURITY.md` 中的方式反馈。
